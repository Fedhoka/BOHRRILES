📥 Importar desde Excel/CSV

Subí los archivos exportados desde whimk.com. El sistema los importa a la base de datos automáticamente.

🖥 Cómo probar localmente (GRATIS)
  1. Descargá e instalá XAMPP (gratuito)
  2. Copiá la carpeta backend/ a htdocs/bohrriles/
  3. Abrí phpMyAdmin → importá schema.sql
  4. Editá config.php con usuario root sin contraseña
  5. Abrí este HTML en el navegador
  6. Probás todo. Cuando estés conforme, subís al hosting
💡 Hosting recomendado
Hostinger — ~$3 USD/mes, incluye PHP + MySQL + dominio
No necesitás VPS. El sistema está optimizado para hosting compartido.
